## üåü Overview
This project is a minimal working prototype designed to enhance conversational AI by providing context-aware follow-up prompts. It analyzes the relationship between a user's original query and the AI's response, identifies critical gaps, and suggests targeted clarifications or elaborations.

The core technology uses a two-stage, prompt-engineered pipeline leveraging the Google AI SDK and the Gemini model to ensure high-quality, actionable analysis.

### **Prerequisites**
- Python **3.8+** installed on your system.

### **Dependencies**
Install the required libraries:

```bash
pip install streamlit google-genai
```
## Execution Steps

### 1. Run the Application
Execute the Streamlit app **`streamlit_app.py`**:

```bash
streamlit run streamlit_app.py
```

### 3. Configure API Key
When the app opens in your browser, enter your **Gemini API Key** in the sidebar to initialize the client.

### 4. Test the Tool
Enter:
- A **User Prompt**
- An **AI Reply**

Then click **Analyze**.
